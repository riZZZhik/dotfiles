# riZZZhik's dotfiles

## Installation

```bash
# Clone dotfiles repo
git clone https://github.com/riZZZhik/dotfiles.git $HOME/.dotfiles

# Go to the dotfiles directory
cd $HOME/.dotfiles

# Install dotfiles
./install
```
